Terms of Use

With great power comes great responsibility

Data Loader can change your data in a big way that may not be possible to reverse if mistakes are made. By using it, you acknowledge that you understand the risks in mass-importing, mass-updating and mass-deleting data in Bullhorn.

Be mindful of others

Data Loader can load millions of records in a very short time from an API user. This can have an impact on actual users of the system. If you plan to load more than 100,000 records at once, please do it off-hours.

For more information

Consult the Data Loader Wiki for more helpful ideas. Report bugs or feature requests through the Data Loader Issues on GitHub.
